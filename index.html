¿Porque usar estos colores en nuestra pagina?

color azul ya que, como lo dice el artículo, se utiliza para transmitir un alto nivel de confianza, ayudara también a que el usuario tenga paz, esto hará que el visitante se sienta seguro, y/o tenga la mentalidad de que el inglés es fácil y puede aprender algo de nuestra página.
El color verde, ya que como dije anteriormente este es otro color que causa seguridad, además que en ocasiones e escuchado que sirve para tener una mejor concentración, lo que hará que el visitante pueda aprender un poco de algún tema de inglés.
el color violeta, el articulo dice que este color se asocia con la espiritualidad y creo que sería bueno para nuestro visitante. El amarillo también decidimos sería un buen color, ya que este color está asociado con la alegría, entre más llamativa sea la página, mejor.
